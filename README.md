# nand2tetris-rtl
An open-source FPGA implementation of the Hack CPU and peripheral suite.

## Academic Integrity Policy

This project is part of an educational implementation of the "Nand to Tetris" course 
(https://www.nand2tetris.org/). 

The course authors (Noam Nisan and Shimon Schocken) request that students and self-learners 
have the benefit and challenge of doing original work. 

- **Please do not post your project solutions publicly** (e.g., in public GitHub repositories, 
  blogs, or forums).
- If you are using this repository to organize a course, please ensure students submit 
  their work in private, password-protected spaces.

Help us ensure that future students can experience the thrill of original work and 
self-discovery.
